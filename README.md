# Halyk HR Service Portal

Единый портал HR-услуг для группы компаний Халык Банк.

## Возможности

- 📋 Каталог HR-сервисов (8 услуг, 5 категорий)
- 📝 Подача заявок с формой и подтверждением
- 📊 Отслеживание статуса заявок (7 этапов workflow)
- 📈 Аналитика и метрики HR Service Center

## Запуск

\`\`\`bash
npm install
npm start
\`\`\`

## Технологии

- React 18
- CSS-in-JS (inline styles)
- Без внешних UI-библиотек

## Структура

\`\`\`
src/
  App.jsx      — основное приложение
  index.js     — точка входа
public/
  index.html   — HTML-шаблон
\`\`\`
